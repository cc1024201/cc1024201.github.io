反转链表

2021-06-08 3 min read

描述

输入一个链表，反转链表后，输出链表的表头

示例1

输入：{1，2，3}
返回值：{3，2，1}

方法一：构造链表

可以先用一个按照插入顺序排序的集合将单链表每个节点都存起来，然后再构造链表
此方法简单易懂

/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Solution {
    public ListNode ReverseList(ListNode head) {
        if (null == head) {
            return null;
        }
        List<ListNode> list = new ArrayList<>();
        while (head != null) {
            list.add(head);
            head = head.next;
        }
        Collections.reverse(list);
        for (ListNode node : list) {
            node.next = null;
        }
        head = list.get(0);
        list.remove(0);
        ListNode currentNode = head;
        while (list.size() > 0) {
            currentNode.next = list.get(0);
            list.remove(0);
            currentNode = currentNode.next;
        }
        return head;
    }
}

方法二：正规解法

此题想考察的的是：如何调整链表指针，来达到反转链表的目的。

  1. 初始化：3个指针
    • pre指向已经反转好的链表的最后一个节点，最开始没有反转，所以为null
    • cur指向待反转链表的第一个节点，最开始第一个节点待反转，所以为head
    • nex指向待反转链表的第二个节点，目的是保存链表，因为cur改变指向后，后面的链表会失效，所以需要保存
  2. 循环执行以下三个操作
    • nex = cur.next, 保存作用
    • cur.next = pre，没有反转的链表的第一个节点的下个节点是已反转链表的最后一个节点（在此完成反转）
    • pre = cur, cur = nex; 指针后移， 操作下一个未反转链表的第一个节点
/*
public class ListNode {
  int val;
  ListNode next = null;

  ListNode(int val) {
      this.val = val;
  }
}*/
public class Solution {
  public ListNode ReverseList(ListNode head) {
      ListNode pre = null;
      ListNode cur = head;
      ListNode nex = null;
      while (null != cur) {
          nex = cur.next; //记录当前节点的下一个节点位置
          cur.next = pre; //让当前节点指向前一个节点位置，完成反转
          pre = cur; //pre往右走
          cur = nex; //当前节点往右走
      }
      return pre;
  }
}
